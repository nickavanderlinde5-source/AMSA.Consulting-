Client Problem & Key Insights

The Challenge

Kumba Iron Ore faces unauthorized entry and illegal mining threats at Sishen. Traditional physical security often causes community friction, threatening the "Social License to Operate."

Stakeholder Insights

High unemployment (31.4%) drives desperation. Stakeholders require a solution that provides safety and partnership rather than exclusion and "Fortress Security."

MVP: The Shared Value Community Reporting Dashboard

A social-digital interface designed specifically for the residents surrounding the Sishen Mine to foster shared security and trust.

Core MVP Features

  • Anonymous Reporting Portal: Residents report suspicious activity securely without fear of retaliation.
  • Interactive Sishen Map: Allows users to accurately "drop a pin" on locations where illegal acts are spotted.
  • POPIA Compliance: Built-in encryption ensuring all user data remains anonymous and legal.
  • Safety Alerts: Real-time notifications to community members about safety risks.

Note: This dashboard can be applied across all companies. Kumba Iron Ore's Sishen Mine used for illustrative purposes.
